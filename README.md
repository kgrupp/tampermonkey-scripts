# TamperMonkey Scripts

## How to Use

1. Install [TamperMonkey](http://www.tampermonkey.net/)
2. Click on the script name to trigger the automated installation process in Tampermonkey.

## Currently Available Scripts

| [youtube-hide-shorts](https://github.com/kgrupp/tampermonkey-scripts/raw/main/youtube-hide-shorts.user.js) | Hides all Shorts in Youtube. |
